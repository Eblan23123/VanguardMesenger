Что мы собираем

  • Никнейм — отображается другим пользователям
  • @Юзернейм — для поиска вас другими
  • Email — обязателен, только для восстановления доступа, нигде не отображается
  • Пароль — хранится локально, нужен для входа после очистки данных
  • Фото профиля — по желанию
  • История сообщений — только в вашем браузере

Важно: Все данные хранятся только в вашем браузере. Никакого сервера — мы физически не можем видеть ваши данные.

О никнейме

Никнейм — имя, которое видят все при переписке. Любое: настоящее или псевдоним. Можно менять в любое время в настройках. Ограничение: до 32 символов.

О Email

Email обязателен — только для восстановления доступа если вы потеряли данные браузера. Не отображается другим пользователям, не используется для рассылок.

О @юзернейме

@Юзернейм — уникальный ID для поиска. Только a–z, 0–9, _. Максимум 20 символов. Один аккаунт на email — нельзя создать второй с тем же адресом.

О входе и пароле

Пароль хранится локально в браузере. При очистке данных браузера вход требует пароль. Восстановление — по email через экран входа.

Безопасность

Данные только на вашем устройстве. Сервер не задействован. Очистка браузера удалит профиль — используйте восстановление по email.

О сервисе

VanguardMessenger — минималистичное пространство для прямого общения. Без алгоритмов, без рекламы, без шума.

Возможности

Личные переписки, групповые чаты, голосовые сообщения, поиск по @юзернейму, профиль с фотографией.
